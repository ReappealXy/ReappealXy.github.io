## 📝 网站介绍

这是一个基于 [Hexo](https://hexo.io/) 框架搭建的个人博客网站，主要用于分享技术文章、学习资料、实用工具和个人项目。网站内容涵盖编程技术、实用工具、学习资源等多个方面。

## 🌟 主要特点

- 🚀 基于 Hexo 静态网站生成器构建
- 📱 响应式设计，适配各种设备
- 🎨 美观的用户界面和良好的用户体验
- 🔍 支持文章搜索和分类浏览
- 🎮 内置贪吃蛇小游戏等互动功能

## 📂 内容分类

网站内容主要分为以下几个类别：

- **学习资料**：分享各类学习资源和教程
- **技术文章**：编程语言（Java、JavaScript、HTML、CSS等）相关文章
- **实用工具**：推荐各种实用的软件和在线工具
- **每日一题**：编程练习和算法题解
- **生活随笔**：个人生活记录和感悟

## 🔧 技术栈

- **前端框架**：Hexo
- **部署平台**：GitHub Pages
- **自定义域名**：www.rexy.xin

## 📌 特色内容

网站包含多种特色内容和工具推荐：

- **AI工具**：AI歌曲生成器、AI漫画生成器等
